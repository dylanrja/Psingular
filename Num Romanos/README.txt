------instrucciones-----
para ejecutarlo basta con abrir el proyecto en un entorno de edicion con un compilador (Java)
o en otro caso desde el simbolo de sistema para esto se debe hacer:

1.-Guarda el programa. Después de usar un editor de texto, como Bloc de notas, para crear tu programa Java, guárdalo con una extensión .java. Para que el archivo se guarde como .java, escribe .java después del nombre del archivo y selecciona "Todos los archivos" en el menú desplegable de las extensiones.

2.-Abre el "Símbolo de sistema" o "Terminal". El acceso a la línea de comandos es ligeramente diferente en Windows o Mac.
Windows: presiona Inicio e introduce cmd. Presiona enter para abrir el "Símbolo de sistema".
Mac: En Finder, presiona la pestaña Ir, selecciona Aplicaciones, selecciona Utilidades y elige Terminal.

3.-
Comprueba que Java esté instalado. introduce java -version en la línea de comandos. En caso de que Java esté instalado, verás un mensaje con la información de la versión de Java que esté instalada.
En caso contrario, es posible que necesites instalar el "Kit de desarrollo de Java" de su página web. Es gratuito y podrás encontrarlo en este enlace: http://www.oracle.com/technetwork/java/javase/downloads/index.html.

4.-Ve a la carpeta correcta. Usa el comando cd seguido del nombre del directorio para cambiar el directorio de trabajo.

5.-Compila el programa. podrás compilar el programa enterduciendo javac nombredearchivo.java en la línea de comandos y presionando "enter".
El "Símbolo de sistema" te informará en caso de que haya errores en el programa o dificultades a la hora de compilar.

6.-Ejecuta el programa. enterduce java "nombredearchivo" y presiona enter.
El programa se ejecutará después de presionar enter.
